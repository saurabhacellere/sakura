Sakura-Editor Macro Reference

S_GoLineTop

機能
行頭に移動(折り返し単位)
構文
void S_GoLineTop ( i1 )  使用例

引数型内容
i1整数数値移動方法指定
設定値
引数 i1 の設定値は次のとおりです。

bit内容
01=行頭のスペースを無視する
1- No Use -
20=非選択状態で移動 / 1=選択状態で移動
30=折り返し単位で移動 / 1=改行単位で移動
補足
PPAのマクロ中では $41 などの 16進数表記や変数・数式・関数の使用も可能だが、MACのマクロ中では 65 などの 10進数の定数表記しか受け付けない。
