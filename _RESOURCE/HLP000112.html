■ソースコード公開
開発中のソースをすべて無償で配布しています。

ソースから実行ファイルをビルドするには、以下のどれかが必要です。
・MS Visual C++ 6.0とMicrosoft Platform SDK
・MS Visual Studio.NET (or 2003, 2005, 2008)
・MS Visual C++ Toolkit 2003とMicrosoft Platform SDK
・MS Visual C++ .NET 2005 Express EditionとMicrosoft Platform SDK
・MS Visual C++ .NET 2008 Express Edition
・Borland C++ Compiler 5.5.1 (C++ Builder含む)とMicrosoft Platform SDK
